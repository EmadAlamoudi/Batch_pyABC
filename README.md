# Batch_pyABC
Batch scripts to allow to dynamically allocate a master/worker nodes for the redis server for pyABC

## start a parallel pyABC run:
run submit_job.sh with the follwing parameters:
* port number
* Number of nodes
* queue name
* job time 
* CPUSPERTASK
* python script
## to kill all running job:
run kill_all.sh
